## Features

- **Component First**: No monolithic libraries. Just code you own.
- **Visual by Default**: Styled for 2025, not 1985.
- **Type-Safe**: Built with TypeScript and React.
- **Automated**: CLI to setup projects and add components instantly.

## Quick Start

```bash
# 1. Initialize a new project
mkdir my-tui && cd my-tui && npm init -y
npm install react ink

# 2. Initialize tomcs
npx tomcs init

# 3. Add all components
npx tomcs add --all

# 4. Start building in src/app.tsx!
```
